Basic HTML5 Template

A simple HTML5 page:

                    
<!doctype html>

<html lang="en">
    <head>
        <meta charset="utf-8">

        <title>The HTML5 Herald</title>
        <meta name="description" content="The HTML5 Herald">
        <meta name="author" content="SitePoint">

        <link rel="stylesheet" href="css/styles.css?v=1.0">

        <!--[if lt IE 9]>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.js"></script>
        <![endif]-->
    </head>

    <body>
        <script src="js/scripts.js"></script>
    </body>
</html>

The Doctype

First we have the Document Type Declaration or doctype. this simply tells the browser what kind of document it is. Since HTML5 it can be defined like this:

                    
<!doctype html>

The html Element

Next there is the html element, which hasn't changed much in HTML5. The lang attribute can bee added, but is not necessary.

                    
<!doctype html>
<html lang="en">

</html>

The head Element

The first line inside the head is the one that defines the caracter encoding. In HTML5 it has been simplified to the bare minimum:

                    
<meta charset="utf-8">

The page title is the only mandatory tag inside head, the rest of the meta tags are optionals.

A key element here is the link tag, and its only required attributes are rel and href, and type is no longer necessary.

                    
<title>The HTML5 Herald</title>
<meta name="description" content="The HTML5 Herald">
<meta name="author" content="SitePoint">

<link rel="stylesheet" href="css/styles.css?v=1.0">

For older IE versions

HTML5 includes a number of new tags, normally this isn't a problem if you're using old browsers, because they are rendered like any other element. But in IE8 and older versions, they are seen by the rendering engine as "unknown elements" and are not rendered as a result. To avoid this issue, the "HTML5 shiv" is added in a script tag surrounded by a conditional comment which is implemented in IE9 and earlier.

                    
<!--[if lt IE 9]>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.js"></script>
<![endif]-->
